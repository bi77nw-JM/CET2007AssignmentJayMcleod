# ConsoleApp1
Steps to set up application:
1. Press <>Code and download the zip file
2. Once Downloaded unzip in a file of your choice.
3. Open the .sln in visual studio 2022
4. Before running, you can check the unit tests by clicking on test at the top
   and pressing run all tests.
5. You can then press start to run the application.

To navigate menus, use the promt provided for details. You can use numbers to navigate.
The purpose of this project is to store, update and search player profiles using JSON files.
